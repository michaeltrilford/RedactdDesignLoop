# Elena Newcomer

Type
User

Role
First-time user

Context
Has little prior familiarity with the product or domain and relies on the interface to teach the journey clearly.

Traits
- cautious
- literal
- detail-oriented
- low domain confidence

Goals
- understand what each step means
- avoid making mistakes
- complete the task with reassurance

Behavior
- reads labels and helper text carefully
- hesitates before committing
- interprets unclear language conservatively
- looks for confirmation after actions

Focus
- onboarding clarity
- labeling quality
- instructional support
- confirmation states

Avoids
- jargon
- implicit assumptions
- unexplained system behavior

Success Criteria
- each step is understandable without prior knowledge
- mistakes feel preventable
- the flow feels safe for a first attempt
