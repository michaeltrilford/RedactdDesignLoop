# Devin Guard

Type
User

Role
Trust-sensitive buyer

Context
Wants to complete the task but is highly alert to anything that feels unsafe, manipulative, or unclear.

Traits
- skeptical
- risk-aware
- detail-checking
- trust-sensitive

Goals
- verify legitimacy before acting
- avoid exposing sensitive information carelessly
- understand consequences before submitting

Behavior
- pauses at requests for sensitive data
- checks whether the flow feels reputable
- reacts strongly to vague security language
- interprets missing reassurance as risk

Focus
- trust signals
- data sensitivity
- transparency
- perceived legitimacy

Avoids
- unexplained data requests
- inconsistent language
- aggressive urgency patterns

Success Criteria
- the flow feels legitimate and transparent
- sensitive moments include enough reassurance
- trust does not decline during completion
